AutoBirthday- An automatic birthday SMS sender.

If you have any contacts with stored birthdays, the application will pull them in. If you don't want a certain contact to get an automatic message, you can turn it off for individual contacts.

New contacts can be loaded with the "load contacts" button in the menu. Updating loaded contacts will have to be reloaded with the "reload contacts" button.

Some features it includes are: Automatic new contact loading, Custom message to send, Custom time to send, Three themes to choose from.

CalcGPA+- A College and High School GPA Calculator

What is the problem with most GPA Calculators? It is that schools use different scales and those calculators do not give you an option to change it.

CalcGPA+ solves this problem. This is a GPA calculator for both college and high school. It lets you select up to six classes. College credits range from one-six credit hours and high school has an option for regular/AP/college credit classes. The best part is the it features a custom scale so you can adjust it to your liking.

ConvertForCPP- Converting Between Data Types Has Never Been Easier.

If you convert between data types often while using C++, it is a bit of a hassle to figure out what function you need to convert. In C#, you can use the Convert class that lets you convert between data types in a much easier and consistent way.

My goal was to make converting easier in C++. Instead of using stoi, atoi, to_string, etc to convert, you can just load this header file and call it using Convert::To(Put type here).
